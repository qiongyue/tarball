Tarball
=======
*******
a GUI tools for creating archives from file path.

Usage
=======
*******
the first three line,start as '#!' is the tool's simple config paramater.Just looks like the following:

    #! 程序根目录: F:\www\
    #! 打包存放目录: F:\www\tar\
    #! name=qiongyue&dir=0
    # 以#开头代表这一行为备注

this tools is useful for the maintenance project.
